Name three good reasons to start a branch in a version control system.
Apart from the obvious answers like using it as a sandbox to experiment in or creating branches for individual features to keep them isolated until you merge them, it is also great to use a branch when you are pulling down from the main master and are unsure of potential merge conflicts. This way if you pull into a branch first, you can handle any merge conflicts should they arise, and not taint the master on your local machine. This comes in quite useful on group projects.
Why is it better to use CSS (rather than JavaScript, if possible) for a basic web page animation?
CSS is for styling, Javascript is for user interaction. In my book, a simple animation falls into the style category. Your browser is also able to process CSS more quickly than Javascript.
What is your favorite project that you've done?
Current Conditions is a Rails site layed out with Foundation utilizing the USGS Water API to provide current conditions for all surface water sites in the USA. The site is searchable by state or keyword, and allows users to log in and subscribe to specific site gauges and have them display in a table on their profile page. Users also have the ability to receive email notifications when subscribed sites rise or fall above a user-specified flow threshold. This was a team project I worked on at Portland Code School.
