Hey there, Needmore!

I'm Charlotte Koh, and these are my answers to your questions.

  1. What is your favorite WordPress function? Why?

    I am a big fan of the flexibility of add_shortcode, specifically because of how it can make it easier for end-users to embed media. This possibility came to my attention when I was attending a conference and sat in on a talk called WordPress FUNctions.

    I work with people who are either unfamiliar or uncomfortable with manipulating code, so having the ability to simplify the process to something as friendly looking as [yourvideo file='video.mp4'] seems pretty great.

  2. Name three good reasons to start a branch in a version control system.

    1. When you want to work on a bug fix
    2. When you want to work on a new feature
    3. When you want to experiment in general without affecting the main files

  3. Why is it better to use CSS (rather than JavaScript, if possible) for a basic web page animation?

    CSS is faster than JavaScript since JavaScript must be interpreted by the browser. Also, since animations could be considered styles for the page (similar to hover states on links), it would make sense to keep animations with the rest of the styles in the CSS file.

  4. What is your favorite project that you've done? (include link and/or image)

    My favorite project is the redesigned University of South Florida website. I began working on it in July 2011, and it launched Dec. 7, 2012. The project gave me the opportunity to dig into responsive design and coding for a large-scale website.

    What really excited me was the ability to make the website much more accessible than it had been in the past. I spent hours reading up on Section 508, WCAG and WAI-ARIA roles, as well as turning off images and JavaScript, navigating the site using only a keyboard, and increasing and decreasing font sizes. It was all worth it when I received an email from a woman who was happy someone working on the Web cared to take her vision needs seriously.

  5. Look at our portfolio and find something we've built that could be improved. Suggest how!

    All of the websites in the Needmore portfolio, with the apparent exception of Savoir Flair, use WordPress. Of the WordPress websites that provide links, only Gist Brands, Stumptown Coffee Roasters and Löyly have branded login screens. (The Needmore website also has a branded login screen).

    I would suggest implementing a branded login screen for clients by default. It’s a small detail, but it can help to tie together the whole experience for users. Also, while I can’t tell if this has been done, it would also be great to carry the branding through to the admin screen. It is possible to change the little WordPress icon next to the website name beneath the top navigation bar.
