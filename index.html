Questions and Answers:

  • What is your favorite WordPress function? Why?

    My favorite WordPress function is register_post_type. This function is one of the main ways Wordpress works as not just a blogging platform, but a content management system. Behind this fairly simple function is the freedom to create different types of posts and to style them using the standard single-post and post-archive pages and to give them their own taxonomies, and that freedom and flexibility is why this function is my favorite.

  • Name three good reasons to start a branch in a version control system.

    1. To keep things organized: starting a branch allows you to delineate specific tasks more easily, such as bug fixes, which can then be folded into the master branch. You can create a branch for one specific task, check to make sure it works, and then add it to the stable, master branch with commit notes, keeping things organized.

  • Why is it better to use CSS (rather than JavaScript, if possible) for a basic web page animation?

    CSS animations are faster than JavaScript because they are rendered by the browser. They require less system resources. They don't require loading scripts and potentially libraries like jQuery, so it will decrease page load and bloat.

    You could also make an argument that using CSS instead of JavaScript for animation works with the principles of progressive enhancement. Any page animation is arguably not an integral part of the website's content. Visitors without modern browsers won't see the CSS animations, but they will benefit from not having to load and run scripts whose only purpose is to provide visual interest.
